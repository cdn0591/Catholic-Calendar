# Catholic-Calendar
天主教日历
天主教香港教区礼仪日历 (2026) 订阅服务

这是一个自动化的日历生成器，数据源自 GCatholic.org。

如何订阅到苹果日历 / Google Calendar

获取原始文件链接：

在此 GitHub 仓库中找到 catholic_hk_2026.ics 文件。繁体版
在此 GitHub 仓库中找到 catholic_cn_2026.ics 文件。简体版
各位按需选择


点击该文件，然后点击 "Raw" 按钮。

复制浏览器地址栏中的 URL（链接应以 raw.githubusercontent.com/... 开头）。

在 iPhone/iPad 上订阅：

打开 设置 (Settings) -> 日历 (Calendar) -> 账户 (Accounts)。

点击 添加账户 (Add Account) -> 其他 (Other) -> 添加已订阅的日历 (Add Subscribed Calendar)。

粘贴刚才复制的 URL。

在 Mac 上订阅：

打开 日历 (Calendar) 应用。

点击菜单栏的 文件 (File) -> 新建日历订阅 (New Calendar Subscription...)。

粘贴 URL 并点击订阅。

自动化说明

脚本通过 GitHub Actions 每周自动运行一次，确保数据与源站同步。

订阅链接一旦添加，日历将根据你的设备设置自动静默更新。
